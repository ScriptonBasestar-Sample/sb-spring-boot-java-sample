# spring-boot-sample
샘플코드 모음

gradle module로 최대한 분할된 프로젝트 기본구조를 생성하려고 하는데

기능적으로 분할하는데에는 한계가 있다.
core,app,domain,client,external정도는 기본적으로 나눠놓고 시작할 수 있다.
- core 라이브러리성
- app 앱서버
- domain hibernate 설정
- client dto
- external 외부연동

프로젝트가 복잡해지면 업무 단위로 모듈을 나눠야 하는데 그런경우 적합하지 않음
account, product, ...
